# GitHub Copilot Agent ワークショップ - スキーショップマイクロサービス編

## 「AI支援による効率的なシステム理解と開発」

### ワークショップ概要

- **時間**: 2時間
- **対象**: 初心者〜上級者（段階的な難易度設定）
- **テーマ**: 9つのマイクロサービスからなるスキーショップシステムの理解と実装
- **目標**: GitHub Copilot Agentを使ってシステム全体を理解し、フロントエンドサービスを実装する

### システム概要

本ワークショップでは、以下の9つのマイクロサービスで構成されるスキーショップシステムを扱います：

1. **Authentication Service** - 認証・認可サービス
2. **User Management Service** - ユーザー管理サービス  
3. **Inventory Management Service** - 在庫管理サービス
4. **Sales Management Service** - 販売管理サービス
5. **Payment Cart Service** - 決済・カートサービス
6. **Point Service** - ポイントサービス
7. **Coupon Service** - クーポンサービス
8. **AI Support Service** - AI支援サービス
9. **API Gateway** - APIゲートウェイ

**総エンドポイント数**: 163個（一般利用者向け: 99個、管理者向け: 64個）

**ストーリー**: フロントエンドアーキテクトとして、2つの異なるWebアプリケーションの設計を理解しよう

**実践内容**:

```text

一般利用者向け、管理者向けのエンドポイントの一覧は all-endpoint-list.md 記載されていますので、そちらを参考にしてください。
```

## 事前準備

### 必要な環境

- Visual Studio Code + GitHub Copilot
- Node.js (v18以上)
- Git
- ブラウザ（Chrome/Firefox推奨）

### 事前インストール

```bash
npm create react-app customer-frontend --template typescript
npm create react-app admin-frontend --template typescript
npm install @mui/material @emotion/react @emotion/styled
npm install @mui/icons-material
npm install axios react-router-dom
npm install recharts  # 管理画面用
```

### プロジェクト構成

```
WorkShop/
├── customer-frontend/     # 一般利用者向け（薄い青色）
├── admin-frontend/        # 管理者向け（薄い赤色）
├── 各種マイクロサービス/
└── all-endpoint-list.md   # API仕様書
```
